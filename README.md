# La Tale SPF Viewer

A tool for viewing and extracting the contents of SPF files, an archive format used in the MMORPG LaTale.

The older tools — SPF_Viewer and SPF_Viewer_Qt — are no longer maintained.

LaTale_SPF_Viewer is their successor: it’s more feature-rich, and requires no compilation.

You saw it right. It’s written in HTML and JavaScript.
